# Vision
Learn programing (and build fun toys) using Raspberry Pi, arduino, esp8266, cli, python3, c...

# Methods
## Hardware
- Raspberry Pi
- esp8266
- arduino sensors
  - PIR (passive IR)
  - IR (active infra red) obsticle detection
  - I2C OLED display
  - Neo Matrix LED matrix
  - capacitative button

## Software
- Linux - Raspberry OS
- python3
- arduino-cli (on Raspberry OS)
- clang

- Inbuilt LED blink

- Capacitive button example
